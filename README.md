# Chain-Reaction
Chain reaction game

### Input Format for moves
Supported formats are as follows:
- \[number] \[number]
- \[letter] \[letter]
- \[number] \[letter]
- \[letter] \[number]

As long as the input format is not the first one, the space can be ignored

PS - The [number] indexing is 1-based indexing.
